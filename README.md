## Course Introduction

The aim of this course is to expose students to the basic principles of computer communication and networking technologies. Upon completion, students will have a strong working knowledge of:

- Network components  
- Transmission links  
- Link control and protocols  
- Network topologies  
- Error detection and correction  
- Network management and security  
- Privacy, legal, and ethical issues

# Weekly Breakdown

## Week 1 – Introduction to Networks & the Internet

- What is the Internet?  
- Protocols & standards (IETF, RFCs)  
- Client–Server vs Peer-to-Peer  
- Enterprise & cloud network models  
- Internet governance & addressing  

## Week 2 – Physical Layer, Media & Network Topologies

- Physical media: copper, fiber, wireless  
- Structured cabling standards (TIA/EIA)  
- Network topologies  
  - Star, Ring, Mesh, Tree  
  - Spine-Leaf, Fat Tree (Data Center)  
  - Hybrid designs  
- Transmission impairments  

## Week 3 – Data Link Layer & Error Detection

- Link layer purpose  
- Framing & error control  
- Parity, Checksum, CRC  
- ARQ protocols  
- MAC addresses  

## Week 4 – Multiple Access & LAN Switching

- Multiple access protocols  
  - ALOHA, Slotted ALOHA  
  - CSMA/CD, CSMA/CA  
  - Token Ring  
- Ethernet standards (802.3)  
- Switch operation & MAC learning  
- VLANs & trunking  

## Week 5 – Network Models & Encapsulation

- OSI vs TCP/IP  
- Encapsulation/Decapsulation  
- Protocol Data Units (PDU)  
- Enterprise role of layers  

## Week 6 – Application Layer Fundamentals

- Network applications essentials  
- Sockets & API  
- HTTP/1.1, HTTP/2, HTTP/3  
- REST API basics  

## Week 7 – DNS & P2P Systems

- DNS architecture & resolution  
- Anycast DNS  
- P2P applications  
  - BitTorrent  
  - Distributed Hash Tables  

## Week 8 – Transport Layer: UDP & Reliable Transfer

- UDP services & limitations  
- Principles of reliable delivery  
- RDT protocols  
- Error & flow control  

## Week 9 – Transport Layer: TCP Deep Dive

- TCP segment format  
- Three-way handshake  
- RTT & timeout estimation  
- Flow control  
- Window scaling  

## Week 10 – Congestion Control & Performance

- Congestion principles  
- TCP congestion control  
  - Tahoe, Reno  
  - Cubic, BBR  
- Explicit Congestion Notification (ECN)  
- Traffic shaping & policing  

## Week 11 – Network Layer & IP Protocols

- Network layer overview  
- IPv4 addressing & subnetting  
- NAT & PAT  
- IPv6 basics  
- ICMP  

## Week 12 – Routing Algorithms & Protocols

- Link State vs Distance Vector  
- RIP, OSPF  
- Hierarchical routing  
- MPLS basics  

## Week 13 – BGP, SDN & Modern Routing

- Border Gateway Protocol (BGP)  
- Routing policy  
- Autonomous Systems  
- Software-Defined Networking (SDN)  
- OpenFlow  

## Week 14 – Network Management & Monitoring

- SNMP  
- NETCONF  
- Telemetry  
- Observability  
- SIEM integration  

## Week 15 – Security & Cryptography in Networks

- Threat landscape  
- Cryptographic primitives  
  - Symmetric / Asymmetric  
- SSL/TLS  
- Firewalls & IDS/IPS  
- Zero Trust  
- VPN technologies
